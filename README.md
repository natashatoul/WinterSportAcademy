# WinterSportAcademy
Backend API + Individual project + ASP.NET + Web API + JWT Authentication+ SQLite
**First step**
mkdir Winter_Sport_Academy - create folder
cd Winter_Sport_Academy - change direction  
dotnet new webapi - generate new ASP.NET Core Web API from the template (Program.cs, Controllers/WeatherForecastController.cs, appsettings.json и т.д.).
code . - open folder in the Visual Studio Code
mkdir Models - create Models (Student, Course, EquipmentRental, Schedule, Instructor, Enrollment)
